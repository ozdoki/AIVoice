# AIVoice

AIVoice は、`Raw` モードと `Polish` モードを切り替えられる Windows 先行の音声入力アプリ構想です。

現時点では実装前の初期リポジトリであり、MVP 計画書をベースに設計を固めています。

## 目的

- `Raw` モードで発話内容をできるだけそのまま入力する
- `Polish` モードで自然発話を読みやすい文章へ整形して入力する
- 共通の会話キーで音声入力を開始し、別キーでモードを切り替える
