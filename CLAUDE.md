# AIVoice

## プロジェクト概要
- Windows 先行の音声入力アプリ。
- `Raw` モードは発話をできるだけそのまま入力する。
- `Polish` モードは自然な文章へ整形して入力する。
- MVP の実装基盤は `Tauri 2 + Rust + Web UI` を前提にする。

## 現在の主要資料
- 一次計画書: `docs/aivoice-mvp-plan.md`
- リポジトリ概要: `README.md`

## セッション開始時の手順
- 実装や設計変更の前に `docs/aivoice-mvp-plan.md` を確認し、既存の MVP 方針と用語を合わせる。
- 実装コードがまだ少ない前提で、ディレクトリ追加や依存追加は最小構成を優先する。
- 長く維持したいルールはここに書かず、`.claude/rules/` の対象ファイル別ルールを参照する。
